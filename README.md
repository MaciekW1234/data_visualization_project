# Data Visualization Project — Marriage, Divorce & Mental Health

A data visualization project exploring the relationship between **marital status, divorce rates, and mental health** across European countries. The final deliverable is a scientific poster combining several visualizations built in R with the `ggplot2` ecosystem.

## About the Project

The poster examines how marital status — and divorce in particular — correlates with the demand for and supply of mental-health care across Europe. It draws on country-level statistics (divorce rates, availability of mental-health specialists), individual-level data on professional care-seeking, and mental-health condition prevalence (depression, anxiety, OCD, insomnia). The work was developed for a Data Visualization Techniques course as a collaborative team project.

The final poster is available in [`marriage/plakat.pdf`](marriage/plakat.pdf).

## Visualizations

The poster contains five hand-crafted charts, each generated by a dedicated R script:

| Script | Visualization | Description |
|---|---|---|
| `barplot_table3.R` | **Bar chart** | Likelihood of seeking professional mental-health care across marital statuses (Married, Divorced, Single, Widowed, Other), expressed as relative risk ratios |
| `mapkiDoTable2.R` | **Choropleth map of Europe** | Number of mental-health specialists per divorce, visualized as a color gradient over a map of Europe |
| `word_island.R` | **Word cloud** | Most common reasons for divorce (No commitment, Affairs, Arguing, Financial problems, …) sized by frequency |
| `wykres4.R` | **Grouped bar chart** | Mean prevalence of depression, anxiety, OCD, and insomnia across countries grouped by divorce-rate category (low / medium / high / very high) |
| `wykresNr3.R` | **Scatter plot** | Predicted mental-health specialist visits as a function of country-level divorce rate, broken down by marital status (Married vs. Divorced/Separated) |
| `countriesMarriage.R` | *Data prep* | Augments the country-level table with additional European countries and a derived `Custom_factor = Care_supply / Divorce_rate` metric |

All charts were exported with transparent backgrounds for clean integration into the final poster.

## Getting Started

### Prerequisites

- R (version 4.0 or higher)
- RStudio (recommended)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/MaciekW1234/data_visualization_project.git
   cd data_visualization_project
   ```

2. Install the required R packages (one-time setup):
   ```r
   install.packages(c(
     "ggplot2", "dplyr", "tidyr", "readr", "scales",
     "maps", "mapdata",
     "wordcloud2", "htmlwidgets", "webshot"
   ))

   # Required for exporting the word cloud as PNG
   webshot::install_phantomjs()
   ```

### Usage

The scripts expect the source CSV files (`Table 1`–`Table 4`, `hospitaladm.csv`, `TeenDepr.csv`) to be available in the working directory. Some scripts reference local absolute paths (e.g. `C:/Users/…/Table2_5.csv`) — update these to point to your local data location before running.

To reproduce a chart, open the corresponding `.R` file in RStudio and source it:
```r
source("marriage/kod/barplot_table3.R")
```
Each script saves its output as a `.png` (or `.html` for the word cloud) into the working directory.

## Data Sources

The project uses tabular data on:
- **Individual-level descriptive statistics** — marital status, gender, care-seeking behavior
- **Country-level descriptive statistics** — divorce rates and mental-health specialist supply per European country
- **HLM Poisson regression results** — coefficients used to model the relationship between marital status, divorce rate, and predicted specialist visits
- **Mental-health condition prevalence** — depression, anxiety, OCD, and insomnia rates by country
- **Hospital administration data**

The raw CSV files are not included in this repository.
